Comment pouvons-nous vous aider ?

Recherchez dans les guides ou parcourez les rubriques ci-dessous

Guides de demarrage

Inscrire un nouveau patient
3 min de lecture

1. Cliquez sur Nouveau patient dans la barre laterale

2. Remplissez les informations : nom, prenom, telephone, quartier

3. Selectionnez la langue preferee du patient

4. Ajoutez un contact de confiance (optionnel mais recommande)

5. Indiquez comment le patient a ete refere

6. Cliquez sur Inscrire le patient — un code unique sera genere

Apres l'inscription, vous pourrez programmer son premier rendez-vous.

Programmer un rendez-vous
2 min de lecture

1. Allez dans Prise de RDV

2. Recherchez le patient par nom ou code

3. Selectionnez le type de visite (consultation, retrait medicaments, bilan sanguin, club)

4. Choisissez la date et un creneau disponible

5. Selectionnez le mode de rappel (SMS, WhatsApp, appel vocal)

6. Cliquez sur Confirmer le rendez-vous

Les rappels seront envoyes automatiquement selon votre configuration.

Gerer un rendez-vous (termine, manque, reporte)
2 min de lecture

Depuis le dashboard : Cliquez sur l'icone a droite du RDV dans la liste "RDV du jour"

Depuis le calendrier : Cliquez sur l'evenement

Une fenetre s'ouvre avec 4 options :

- Termine : ajoutez vos notes de consultation et programmez le prochain RDV

- Manque : choisissez une action (rappel SMS, assigner un ASC, reprogrammer)

- Reporte : selectionnez une nouvelle date et la raison

- Annule : le RDV est retire du calendrier

Suivre un patient en retard (module ASC)
3 min de lecture

1. Les patients en retard apparaissent automatiquement dans le module ASC

2. Ils sont tries par urgence (rouge = plus de 21 jours, orange = 7-21 jours)

3. Cliquez sur Rapport pour remplir le formulaire de visite

4. Indiquez si le patient a ete trouve, la raison de l'absence

5. Reprogrammez un RDV si le patient accepte

La carte des zones affiche la localisation des patients a visiter.

Rappels et SMS

Comment configurer les rappels SMS ?
Allez dans Parametres, section Preferences de rappels. Activez ou desactivez les rappels J-7, J-2 et jour J. Pour configurer le fournisseur SMS, allez dans Parametres et relancez l'assistant de configuration SMS.
Un SMS n'a pas ete livre, que faire ?
Verifiez dans Rappels le statut "Echec". Les causes frequentes : numero incorrect, telephone eteint, credit API epuise. Vous pouvez renvoyer le rappel manuellement ou contacter le patient par un autre canal.
Les messages mentionnent-ils la maladie ?
Jamais. Tous les messages sont generiques : "rappel de votre rendez-vous sante". Vous pouvez personnaliser les modeles dans Rappels > Modeles de messages mais aucune mention de pathologie ne doit etre ajoutee.

Gestion des patients

Comment declarer un deces ou une sortie ?
Ouvrez la fiche du patient, cliquez sur Sortie du programme. Choisissez le motif (deces, transfert, abandon, perdu de vue, guerison), la date et ajoutez des notes. Le dossier sera archive.
Que signifie le score de risque ?
Le score de risque (1 a 10) estime la probabilite qu'un patient manque son prochain RDV. Il est calcule a partir de : nombre de RDV manques, distance au centre, duree sous traitement, tranche d'age. Plus le score est eleve, plus le suivi doit etre rapproche.
Comment changer la frequence de suivi ?
Lors de la mise a jour d'un RDV (statut "Termine"), vous pouvez choisir la frequence : mensuel, trimestriel ou semestriel. Le prochain RDV sera programme automatiquement selon cette frequence.

Technique et parametres

MaSante fonctionne-t-il sans internet ?
Oui. Toutes les fonctions (RDV, fiches patients, rapports) fonctionnent hors ligne. Seuls les rappels SMS necessitent une connexion. Les donnees se synchronisent automatiquement quand internet revient.
Comment ajouter un nouvel utilisateur ?
Seul l'administrateur peut creer des comptes. Allez dans Parametres > Equipe soignante, cliquez sur Ajouter, remplissez les informations et attribuez un role (medecin, infirmier, ASC, admin).
Comment sauvegarder les donnees ?
MaSante sauvegarde automatiquement toutes les donnees en local. Pour une sauvegarde externe, allez dans Parametres et utilisez la fonction Exporter la base. Nous recommandons une sauvegarde hebdomadaire sur cle USB.

Depannage

L'application est lente, que faire ?
Verifiez dans Parametres > Systeme l'utilisation de la base de donnees. Si vous approchez 2 000 patients avec 5+ utilisateurs, contactez-nous pour une migration vers PostgreSQL. Sinon, essayez de redemarrer l'application.
J'ai oublie mon mot de passe
Contactez l'administrateur de votre centre. Seul l'admin peut reinitialiser un mot de passe. Si vous etes l'admin, utilisez la procedure de recuperation par email configuree lors du setup initial.
Comment contacter le support ?
MaSante est un logiciel open source gratuit. Pour obtenir de l'aide : consultez la documentation sur masante.africa, rejoignez la communaute, ou contactez-nous pour un contrat de support.
